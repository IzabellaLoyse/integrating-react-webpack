# 🛠️ Integrating React Webpack

- Aplicação React feita com Webpack

## ⚙️ Rodando a aplicação

- Para rodar o server utilizando o NPM

`npm run dev`

- Para rodar o server utilizando o Yarn

`yarn dev`

## ⚙️ Build da aplicação

- Build com NPM

`npm run build`

- Build com Yarn

`yarn build`
